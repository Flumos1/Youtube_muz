# ElevenLabs — настройки для Эпизода 2 (тёмный / медленный / серьёзный)

> Файл озвучки: `Ep2_VO_ElevenLabs_EN.txt` — вставляй ЕГО, без скобок-тегов.
> Эти настройки = твои теги [SERIOUS]/[CALM]/[DARK]/[SLOW], но средствами, которые движок реально слушает.

## Модель
- **Eleven Multilingual v2** — лучшая для выразительного длинного нарратива.
  (Turbo v2.5 — если важнее скорость/цена, но v2 звучит «дороже».)

## Слайдеры (Voice Settings)
| Параметр | Значение | Зачем |
|----------|----------|-------|
| **Stability** | **50–55%** | Тёмный, ровный, контролируемый тон. Ниже 40% — «гуляет»; выше 65% — становится плоским/роботичным. |
| **Similarity** | **78%** | Близко к референсу голоса, без артефактов. |
| **Style exaggeration** | **5–10%** | Низко = серьёзно и сдержанно. Высокий Style переигрывает и делает запись нестабильной. |
| **Speaker boost** | **ON** | Чище и плотнее голос. |
| **Speed** | **0.90** | Это твой [SLOW]. Медленнее естественной речи, но ещё не «вязко». Можно 0.88. |

## Как зашиты твои теги (без скобок)
- **[PAUSE] между блоками** → пустая строка между абзацами. ElevenLabs делает там паузу автоматически. В файле 9 блоков = 8 пауз.
- **[SLOW] / [QUIET] моменты** → многоточия `…` (например «So he leans over... and spits»). Многоточие = заминка/придых.
- **[EMPHASIS] / [HEAVY]** → короткие рубленые фразы на отдельных строках («Roger was miserable.», «Five months.», «Every member except one.»). Точка между ними = удар.
- **[DARK] / [SERIOUS]** → задаётся Stability+Style выше, на весь текст.

## Если хочешь паузы ДЛИННЕЕ (опционально)
Между блоками можешь вручную вставить тег паузы — он работает на v2/Turbo и НЕ читается вслух:
```
<break time="1.0s" />
```
Не злоупотребляй: больше ~3 сек или много подряд → движок начинает «заикаться».

## ВАЖНО про теги в скобках
- НЕ вставляй `[SERIOUS]`, `[DARK]`, `[CALM]` и т.п. в поле ввода обычной модели — прочитает вслух или проигнорирует.
- Реальные теги подачи понимает только модель **v3 (alpha)** и только из своего словаря: `[whispers]`, `[sighs]`, `[exhales]`, `[sarcastic]` и т.д.
- Если решишь тестировать v3 — скажи, я пересоберу файл с настоящими v3-тегами в нужных местах (шёпот на тёмных строках, выдох перед ударными выводами).
