# Channel Rebrand — Flumos

**Verdict:** Rebrand. The old "Piano Cover by Flumos" + treble-clef avatar mismatches the music-history-drama content and confuses both viewers and the algorithm. Keep the **name "Flumos"** (short, brandable) and the **@flumos** handle — just reposition the visuals + About.

## New assets (ready to upload)
- **Banner:** `branding/banner.png` — 2560×1440, text inside the safe area. "FLUMOS / THE DARKEST STORIES IN MUSIC / The hidden history behind the songs".
- **Avatar:** `branding/avatar.png` — 800×800, white cracked-vinyl + lightning emblem on black (reads at small size, YouTube circle-crops it cleanly).

## Where to set it (YouTube Studio)
- **Customize channel → Branding** → Picture = `avatar.png`, Banner image = `banner.png`.
- **Customize channel → Basic info** → Name = `Flumos`, Handle = `@flumos`, paste Description below, add links.
