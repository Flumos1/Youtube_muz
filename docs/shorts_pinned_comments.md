# Shorts — закреплённые комментарии (paste-ready)

**Зачем:** ссылка в комментарии Shorts кликабельна (в описании Shorts — нет). Закреп + «Видео по теме»
+ кнопка в кадре = максимум переходов Short → длинное видео. Вопрос в конце = вовлечение → подписки.

**Как:** открой Short на youtube.com или в приложении → напиши коммент → отправь → «…» → Закрепить.
Порядок приоритета — сверху (самый трафик). Начни с Fleetwood Mac (1.7–1.8K просмотров).

**Шаблон комментария:**
```
🎬 Full story → {LINK}
🔔 Subscribe for a new music story every week.
Which artist should I cover next? 👇
```
