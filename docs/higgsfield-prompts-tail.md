# Higgsfield Prompt Pack — Tail (Beat 4 → Outro) + Replacements

**Model:** Soul Location · **Format:** 16:9 · Each prompt ends with the shared style suffix.  
**COPYRIGHT-CRITICAL:** No real faces. Use silhouettes, hands, objects, shadows, backs-of-heads only. No real album cover, no real award trophy, no logos/text.

**Style suffix (append to every prompt):**
```
, cinematic, analog film grain, moody low-key lighting, 1970s california aesthetic, warm amber tones, photorealistic, shallow depth of field, no text, no logos, 16:9
```

**Negative prompt (every image):**
```
text, words, letters, logos, watermarks, modern technology, visible faces, recognizable people, glitch effect, chromatic aberration, broken glass overlay, cgi look, cartoon
```

> **Naming:** filenames below are PROVISIONAL (B4-01 …). Real audio is 5:21 and faster than the script estimate, so we assign true MMSS codes once we have anchor timestamps. In CapCut, place each shot at the moment its narrated line is spoken — don't force a rigid 5s.
